# AI x Web3 School Learning Journal

Personal learning journal and proof-of-work for AI x Web3 School.

## Links

- [AI x Web3 School Handbook](https://aiweb3.school/zh/handbook/)
- [WCB course page](https://web3career.build/zh/programs/AI-Web3-School)
- [WCB Learning page](https://web3career.build/zh/programs/AI-Web3-School#tab=learning)

## Structure

- `profile.md`: learner profile and goals
- `learning-plan.md`: current learning plan
- `daily/`: daily notes and check-in drafts
- `tasks/`: course task notes
- `experiments/`: small exercises and prototypes
- `handbook-feedback/`: actionable Handbook feedback
- `hackathon/`: project and hackathon material
- `submissions/`: submission records and links
- `templates/`: reusable note templates

## Public Repository Safety

This repository is intended to be public. Never commit API keys, passwords,
verification codes, wallet seed phrases, private keys, private contact details,
internal meeting links, or another person's personal data.
