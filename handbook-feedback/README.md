# Handbook Feedback Workflow

Use this folder for unclear concepts, errors, outdated resources, missing
examples, translation issues, and structural suggestions.

## Workflow

1. Record the exact Handbook page URL.
2. Describe what was confusing or incorrect.
3. Explain the impact on a beginner.
4. Suggest a concrete revision or example.
5. Add evidence or source links when applicable.
6. Review the note to remove private information.
7. Commit it to this public learning repository.
8. Submit it through the Handbook contribution link or upstream GitHub repo.
9. Add the issue or pull-request URL to the feedback note.

## Upstream Links

- [Contribution guide](https://aiweb3.school/zh/contribution/)
- [Handbook repository](https://github.com/lxdao-official/aiweb3school)
