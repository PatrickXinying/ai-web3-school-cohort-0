# Four-Week Learning Plan

This is a lightweight starting plan for 30 minutes per day. The WCB Learning
page takes priority when it assigns a course, meeting, or task for the day.

## Daily Rhythm

- 15 minutes: read one Handbook section
- 10 minutes: explain the concept in your own words or complete a small exercise
- 5 minutes: update the daily note and prepare the WCB check-in

## Week 1: Shared Vocabulary

- Day 1: What AI x Web3 means and how to use the Handbook
- Day 2: LLM basics
- Day 3: Prompt basics
- Day 4: Context basics
- Day 5: Blockchain network basics
- Day 6: Wallet, account, public key, and private key
- Day 7: weekly review and glossary cleanup

## Week 2: How Systems Work

- RAG and reliable information retrieval
- AI Agent and tool use
- Cryptography and digital signatures
- Smart contracts
- Account abstraction
- Weekly comparison: AI Agent permissions vs wallet permissions

## Week 3: AI x Web3 Bridge

- Chain-aware context
- Web3 tool use
- Agent workflows
- Agent wallets and permission boundaries
- AI security
- Draft one small project problem statement

## Week 4: Proof of Work

- Select one beginner project
- Define user, problem, inputs, outputs, and safety boundary
- Build a non-financial prototype or written workflow
- Document decisions and limitations
- Publish a short project summary
- Review the month and choose the next learning track

## Completion Evidence

At the end of four weeks, this repository should contain:

- at least 20 daily notes
- a personal AI/Web3 glossary
- at least 3 small exercises
- at least 1 Handbook feedback item
- 1 documented beginner project or prototype
